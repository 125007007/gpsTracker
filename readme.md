```
cd gpsTracker
```

* Install dependencies
```
npm install
``` 

### Executing program

* Run setup.js | creates output.json file with dummy object
```
node setup.js
```
* Run app.js | logs to output.json and runs live server
```
node app.js
```
* Run app1.js | runs live server with no logging
```
node app1.js
```
